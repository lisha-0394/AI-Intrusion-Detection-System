# AI-Based Network Intrusion Detection System

## Overview
This project is an AI-powered system designed to detect cyber-attacks in network traffic using Machine Learning and Deep Learning models. It identifies malicious activities and enables real-time intrusion detection.

---

## Tech Stack
- Python  
- Flask  
- Scikit-learn  
- TensorFlow / Keras  
- Pandas, NumPy  

---

## Features
- Data preprocessing and cleaning  
- Handling imbalanced data using SMOTE  
- Trained Machine Learning (Random Forest) and Deep Neural Network models  
- Comparative analysis of model performance (accuracy, speed, robustness)  
- Real-time intrusion detection using a Flask web application  

---

## Dataset
- Contains 225,000+ network traffic records  
- Includes 79 features for analysis  
- Dataset not included if file size is large
